# Cvičný projekt – C# konzolová aplikace

Tento projekt slouží jako ukázka práce s verzovacím systémem Git a platformou GitHub.  
Cílem projektu je demonstrovat vytváření verzí programu pomocí commitů a jejich publikování do vzdáleného repozitáře.

## Účel programu

Program je jednoduchá konzolová aplikace napsaná v jazyce C#.  
Po spuštění vypisuje text do konzole.  
V jednotlivých verzích projektu byly postupně přidávány další výpisy, aby bylo možné sledovat historii změn pomocí verzovacího systému.

Projekt tedy slouží především k procvičení:
- práce s Gitem (commit, push)
- správy verzí
- práce s repozitářem na GitHubu

## Jak program spustit

1. Otevřete projekt v prostředí Visual Studio  
   nebo použijte příkazový řádek s nainstalovaným .NET SDK.

2. Zkompilujte a spusťte aplikaci.
